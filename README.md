### 1.Virtual DOM:
- This is a lightweight representation of real DOM 
- There are 2 virtual DOMs
1. Copy of the real DOM
2. Manage the changes

- when two virtual DOMs gets compared with each other the process is known as Diffing Algorithm.
- And finally when changes finalised the copy of the real DOM(V1) update in the Real DOM and this process is known as Reconcilation.
